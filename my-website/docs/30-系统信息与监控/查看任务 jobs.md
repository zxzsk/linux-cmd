## Linux `jobs` 命令

### 1. 简介

`jobs` 用于显示当前 Shell 会话中任务列表，常与 `bg` 和 `fg` 配合使用。

---

### 2. 语法

```bash
jobs [选项]
```
- **无选项**：列出所有任务。  
- `-l`：显示任务 PID。  
- `-n`：显示最近状态变化的任务。  
- `-p`：仅显示任务的 PID。

---

### 3. 常用示例  

1. **列出当前任务**  
   ```bash
   jobs
   ```
   输出：
   ```
   [1]+  Running    ping google.com &
   [2]-  Stopped    top
   ```

2. **显示任务 PID**  
   ```bash
   jobs -l
   ```
   输出：
   ```
   [1]+  12345 Running    ping google.com &
   [2]-  67890 Stopped    top
   ```

3. **仅显示任务 PID**  
   ```bash
   jobs -p
   ```
   输出：
   ```
   12345
   67890
   ```

---

### 4. 快速参考  

| 功能               | 命令          | 说明                  |
|--------------------|--------------|---------------------|
| 列出所有任务       | `jobs`       | 显示任务编号和状态    |
| 显示任务 PID       | `jobs -l`    | 显示任务编号及进程号  |
| 最近状态变化任务   | `jobs -n`    | 显示状态有变化的任务  |
| 仅显示任务 PID     | `jobs -p`    | 列出任务的 PID        |
